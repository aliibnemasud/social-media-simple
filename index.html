01. How does JavaScript Work?

Javascript is an interpreted language. It will work connected with dom on the browser. a browser reads over the JavaScript code each line from top to bottom. Javascript needs an engine to run on the browser. Normally on chrome browser has a v8 engine to run the java script and different browser has different engines to run the javascript. Without a javascript engine, it can’t run on the browser.

02. The difference between Session storage vs Local storage

>> Session storage doesn’t store data when the browser/tap is closed. But the Local storage stores data when the browser/tap is closed.
>> Session storage creates new sessions by opening new taps, with the same link or different. But local storage only stores specific data from users and it is stored on the client-side.
